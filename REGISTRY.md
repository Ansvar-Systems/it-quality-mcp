# Registry Listing

## Short Description

IT quality framework data (ITIL 4, COBIT 2019, ISO 20000, CMMI) via MCP.

## Medium Description

IT Quality MCP server providing structured access to 12 IT quality, governance, and service management frameworks: ITIL 4, COBIT 2019, ISO 20000, ISO 22301, ISO 25010, ISO 38500, ISO 12207, ISO 15288, CMMI v2.0, FitSM, DORA Metrics, and DAMA-DMBOK. Includes 368 practices, 52 SLA metric templates, 111 cross-framework mappings, and 17 maturity level definitions. Part of the [Ansvar MCP Network](https://ansvar.ai/mcp).
