## What's NOT Included

- **PRINCE2** -- Project management methodology (different domain)
- **TOGAF** -- Enterprise architecture framework (planned for future)
- **Six Sigma** -- Quality management methodology (planned for future)
- **ISO 27001** -- Already covered by [security-controls-mcp](https://github.com/Ansvar-Systems/security-controls-mcp)
- **NIST CSF / 800-53** -- Already covered by security-controls-mcp
- **ISO 9001** -- General quality management (not IT-specific)

## Limitations

- **Metadata only** -- The database contains structured metadata (practice names, purposes, KPIs, RACI matrices) written as original summaries. No copyrighted standard text is reproduced.
- **Not a substitute** -- This data supports research and gap analysis. It does not replace purchasing and reading the authoritative standard documents.
- **Coverage depth varies** -- Some frameworks (ITIL 4, COBIT 2019) have richer metadata (inputs, outputs, KPIs, RACI) than others.

## Data Freshness

All framework data is manually curated and reviewed on a quarterly basis. The `check_data_freshness` tool reports per-framework staleness against a 180-day threshold. Last full review: 2026-02-25.
